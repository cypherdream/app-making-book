# Android & Backend Engineering Roadmap

A rigorous, 14-day technical curriculum focused on native Android development (MVVM/Clean Architecture) and scalable Node.js backend integration. 

## Engineering Standards
- **Architecture:** MVVM + Clean Architecture + Repository Pattern.
- **Language:** Kotlin (Android) / TypeScript (Backend).
- **Toolchain:** JDK 17, Android Studio Jellyfish, Gradle, Git, Node.js (LTS).
- **Philosophy:** Zero marketing fluff. Code-first, documentation-second.

## Curriculum Structure
| Phase | Focus |
| :--- | :--- |
| **Week 1** | Native Android Client Engineering |
| **Week 2** | Backend Infrastructure & API Engineering |

## Prerequisites
- Android Studio installed with SDK API Level 34.
- Git configured with Personal Access Token authentication.
- Node.js runtime environment configured.
